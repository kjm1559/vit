Vision Transformer(ViT)
===

This is ViT(Vision Transformer) by keras.

## Usage  
```bash
python run.py
```

## Result  
- Pre-train -> CIFAR10  
- Fine tune -> MNIST  
- Epoch : 20

|type|accuracy|
|:---|-------:|
|only MNIST|96.61|
|Fine tune|96.91|
